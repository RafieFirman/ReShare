# ReShare

Reshare adalah sebuah wadah untuk berbagi dan berdonasi barang-barang bekas yang masih layak pakai.
Dengan kehadiran ReShare diharapkan dapat membantu saudara kita yang membutuhkan serta mengurangi limbah rumah tangga.

## Web ini dibangun dengan :
- Front end : HTML, css, Javascript dan tailwind
- Back end : PHP
- Database : Sql

- Website ini bisa di akses di : [reshare.my.id](https://reshare.my.id/reshare/frontend/index.php)

## Fitur
- Registrasi & Login
- Berdonasi dengan menambahkan foto serta detail barang seperti nama, kondisi dan kategori
- Melihat katalog & filter berdasarkan kategori (rumah tangga, buku, elektronik & pakaian)
- Mengirim permintaan untuk mengambil barang dengan menyertakan alasan
- Donatur bebas memilih kepada siapa barang akan di berikan
- Fitur Leaderboard & Badge bagi orang dengan jumlah donasi terbanyak
- Event
