#ReShare

Reshare adalah sebuah wadah untuk berbagi dan berdonasi barang-barang bekas yang masih layak pakai.
dengan kehadiran ReShare diharapkan dapat membantu saudara kita yang membutuhkan serta mengurangi limbah rumah tangga.

##Web ini dibangun dengan :
Front end : HTML, css, Javascript dan tailwind
Back end : PHP
Database : Sql

Website ini bisa di akses di : reshare.my.id
